Exercício 4
===========

Conversão binário-decimal

a. 0.001
```
1
+- 1 * 2^(-3) = 1 * 0.125 = 0.125
                          -------
                            0.125
```

b. 1.1
```
11
|+- 1 * 2^(-1) = 1 * 0.5 = 1
+-- 1 * 2^(0)  = 1 * 1   = 0.5
                         -----
                           1.5
```

c. 1.0101
```
10101
||||+- 1 * 2^(-4) = 1 * 0.0625 = 0.0625
|||+-- 0 * 2^(-3) = 0 * 0.125  = 0
||+--- 1 * 2^(-2) = 1 * 0.25   = 0.25
|+---- 0 * 2^(-1) = 0 * 0.5    = 0
+----- 1 * 2^(0)  = 1 * 1      = 1
                               --------
                                 1.3125
```

d. 10.01
```
1001
|||+--- 1 * 2^(-2) = 1 * 0.25   = 0.25
||+---- 0 * 2^(-1) = 0 * 0.5    = 0
|+----- 0 * 2^(0)  = 0 * 1      = 0
+------ 1 * 2^(1)  = 1 * 2      = 2
                                -------
                                  2.25
```

e. 1001.001
```
1001001
||||||+- 1 * 2^(-3) = 1 * 0.125 = 0.125
|||||+-- 0 * 2^(-2) = 0 * 0.25  = 0
||||+--- 0 * 2^(-1) = 0 * 0.5   = 0
|||+---- 1 * 2^(0)  = 1 * 1     = 1
||+----- 0 * 2^(1)  = 0 * 2     = 0
|+------ 0 * 2^(2)  = 0 * 4     = 0
+------- 1 * 2^(3)  = 1 * 8     = 8
                                -------
                                  9.125
```

Exercício 6
===========

Erro absoluto e relativo

a. x1 + x2
```
Resultado real:
x1 = 0.834456
x2 = 2.634305
x1 + x2 = 0.834456 + 2.634305 = 3.468761

Considerando um sistema que opera por truncamentos,
x1 = 0.8344
x2 = 2.634

x1 --> 0.8344
x2 --> 2.634  +
       ------
       3.4684

Erro absoluto: 3.468761 - 3.4684 = 0.000361
Erro relativo: 0.000361 / 3.468761 ~=
              ~= 0.000104071742042... ~= 1e-4
```

b. x2 x3
```
Resultado real:
x2 = 2.634305
x3 = 0.103457
x2 * x3 = 0.272537292385

Considerando um sistema que opera por truncamentos,
x2 = 2.634
x3 = 0.1034

x2 --> 2.634
x3 --> 0.1034 *
       ------
       0.2723556 ~
	   0.2723

Erro absoluto: 0.272537292385 - 0.2723 = 0.000237292385
Erro relativo: 0.000237292385 / 0.272537292385 ~=
              ~= 0.0008706785883... ~= 8e-4
```

c. x1 + x4 / x2
```
Resultado real:
x1 = 0.834456
x2 = 2.634305

x1 =  0.8344

Erro relativo: -0.000226941800588... / 6.44517305819941... ~=
              ~= -0.0000352111259914207... ~= -3e-5
```
