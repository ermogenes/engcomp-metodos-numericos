Exercício 2
===========

Conversão binário-decimal

a. 1
```
1
+- 1 * 10^0 = 1 * 1  =  1
                      ----
                         1
```

b. 101
```
101
||+- 1 * 2^0 = 1 * 1 = 1
|+-- 0 * 2^1 = 0 * 2 = 0
+--- 1 * 2^2 = 1 * 4 = 4
                     ---
                       5
```

c. 1000
```
1000
|||+- 0 * 2^0 = 0 * 1 = 0
||+-- 0 * 2^1 = 0 * 2 = 0
|+--- 0 * 2^2 = 0 * 4 = 0
+---- 1 * 2^2 = 1 * 8 = 8
                     ----
                        8
```

d. 1011
```
1011
|||+- 1 * 2^0 = 1 * 1 = 1
||+-- 1 * 2^1 = 1 * 2 = 2
|+--- 0 * 2^2 = 0 * 4 = 0
+---- 1 * 2^2 = 1 * 8 = 8
                     ----
                       11
```

e. 111001
```
111001
|||||+- 1 * 2^0 = 1 * 1  =  1
||||+-- 0 * 2^1 = 0 * 2  =  0
|||+--- 0 * 2^2 = 0 * 4  =  0
||+---- 1 * 2^2 = 1 * 8  =  8
|+----- 1 * 2^2 = 1 * 16 = 16
+------ 1 * 2^2 = 1 * 32 = 32
                         ----
                           49
```
