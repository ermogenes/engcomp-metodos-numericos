Exercício 2
===========

Conversão binário-decimal

a. 1
```
1
+- 1 * 10^0 = 1 * 1  = 1
                     ---
                       1
```

b. 101
```
101
||+- 1 * 2^0 = 1 * 1 = 1
|+-- 0 * 2^1 = 0 * 2 = 0
+--- 1 * 2^2 = 1 * 4 = 4
                     ---
                       5
```

c. 1000
```
1000
|||+- 0 * 2^0 = 0 * 1 = 0
||+-- 0 * 2^1 = 0 * 2 = 0
|+--- 0 * 2^2 = 0 * 4 = 0
+---- 1 * 2^3 = 1 * 8 = 8
                      ---
                        8
```

d. 1011
```
1011

||+-- 1 * 2^1 = 1 * 2 = 2
|+--- 0 * 2^2 = 0 * 4 = 0
+---- 1 * 2^3 = 1 * 8 = 8
                      ---
                       11
```

e. 111001
```
111001

||+---- 1 * 2^3 = 1 * 8  =  8
```

Exercício 4
===========

Conversão binário-decimal

a. 0.001
```
1
+- 1 * 2^(-3) = 1 * 0.125 = 0.125
                          -------
                            0.125
```

b. 1.1
```
11
|+- 1 * 2^(-1) = 1 * 0.5 = 1
+-- 1 * 2^(0)  = 1 * 1   = 0.5
                         -----
                           1.5
```

c. 1.0101
```
10101
||||+- 1 * 2^(-4) = 1 * 0.0625 = 0.0625
|||+-- 0 * 2^(-3) = 0 * 0.125  = 0
||+--- 1 * 2^(-2) = 1 * 0.25   = 0.25
|+---- 0 * 2^(-1) = 0 * 0.5    = 0
+----- 1 * 2^(0)  = 1 * 1      = 1
                               --------
                                 1.3125
```

d. 10.01
```
1001
|||+--- 1 * 2^(-2) = 1 * 0.25   = 0.25
||+---- 0 * 2^(-1) = 0 * 0.5    = 0
|+----- 0 * 2^(0)  = 0 * 1      = 0
+------ 1 * 2^(1)  = 1 * 2      = 2
                                -------
                                  2.25
```

e. 1001.001
```
1001001
||||||+- 1 * 2^(-3) = 1 * 0.125 = 0.125
|||||+-- 0 * 2^(-2) = 0 * 0.25  = 0
||||+--- 0 * 2^(-1) = 0 * 0.5   = 0
|||+---- 1 * 2^(0)  = 1 * 1     = 1
||+----- 0 * 2^(1)  = 0 * 2     = 0
|+------ 0 * 2^(2)  = 0 * 4     = 0
+------- 1 * 2^(3)  = 1 * 8     = 8
                                -------
                                  9.125
```
